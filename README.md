# рџ“– About the Project

Cloud Bus Pass System is a full-stack web application developed to simplify the traditional bus pass management process. The system enables users to register, apply for bus passes, track application status, download approved passes, and print them digitally.

Administrators can securely manage applications, approve or reject requests, maintain user records, export application data, and monitor the complete system through an interactive dashboard.

The project aims to eliminate paperwork, improve efficiency, and provide a secure cloud-based solution for bus pass management.
